# Hermes Agent — 使用、配置与运维笔记

*Hermes 特定的使用约定、配置细节和项目特定的运维协议。*

## 模型选择
- 不要为常规任务使用过于智能的模型。`qwen3.6-plus` 已经是大材小用。
- `flash` 级别模型应该足以应对大多数代理工作——更快更便宜。
- 经过强后训练的小模型（10B 级别）可以高效处理复杂的代理任务。

→ 详细的模型选型矩阵和迁移决策，详见 [LLM 过度思考批判](llm-overthinking-critique.md) 和 [技术复盘备忘录](qwen-to-mimo-migration.md)

## 行为模式
- 带用户前缀的对话式消息（如 `O: ...`）通常需要：分析观点、填补省略的逻辑、验证推理的正确性。
- 用于恢复的会话记录：`# <<<` = 用户消息，`# >>>` = 助手响应。
